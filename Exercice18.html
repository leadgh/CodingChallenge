Exercice 18 - Tableau

Créer un tableau de données comportant une liste de prénoms. Exemple: Julie, Cedric, Jean-Marc, ...

  • Ajouter un bouton vous permettant d'afficher le premier prénom du tableau.
  • Ajouter un bouton vous permettant d'afficher tous les prénoms.
  • Ajouter un bouton vous permettant d'afficher le dernier resultat du tableau.
  • Ajouter la variable "Céline" à la fin du tableau (.push) et afficher le nouveau résultats.
  • Créer un nouvelle instance du tableau, puis supprimer Cedric et Jean-Marc. Intégrer ensuite un bouton vous permetant d'afficher le résultat sur votre navigateur.
